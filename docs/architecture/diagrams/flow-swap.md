# Swap Flow Diagram

## Overview

The swap flow allows users to exchange one token for another using
DEX aggregators.

## State Machine

```
┌─────────┐     ┌──────────┐     ┌───────────┐     ┌──────────┐
│  IDLE   │────▶│ QUOTING  │────▶│ REVIEWING │────▶│ SWAPPING │
└─────────┘     └──────────┘     └───────────┘     └──────────┘
     ▲               │                │                  │
     │               │                │                  │
     │    quote      │                │                  │
     │    error      │                │                  │
     │◀──────────────┘                │                  │
     │                                │                  │
     │         user cancels           │                  │
     │◀───────────────────────────────┘                  │
     │                                                   │
     │                         swap error                │
     │◀──────────────────────────────────────────────────┤
     │                                                   │
     │                         swap success              │
     │◀──────────────────────────────────────────────────┘
                                      │
                               ┌──────┴──────┐
                               │   SUCCESS   │
                               └─────────────┘
```
