## Redux State Flow

```
sendPayment action dispatched
         │
         ▼
┌─────────────────┐
│   send/saga.ts  │
│                 │
│  sendPaymentSaga│
└────────┬────────┘
         │
         ├──▶ setSendStatus('preparing')
         │
         ├──▶ call(prepareTransaction)
         │         │
         │         └──▶ encodeTransferData
         │         └──▶ estimateGas
         │
         ├──▶ setSendStatus('signing')
         │
         ├──▶ call(signAndSend)
         │         │
         │         └──▶ getPrivateKey
         │         └──▶ walletClient.sendTransaction
         │
         ├──▶ setSendStatus('pending')
         │
         ├──▶ call(waitForReceipt)
         │
         ├──▶ setSendStatus('success')
         │
         └──▶ addTransaction({ hash, ... })
```
