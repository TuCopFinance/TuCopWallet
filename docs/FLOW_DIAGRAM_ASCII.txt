┌────────────────────────────────────────────────────────────────────────────────────┐
│ PANTALLA: TransactionSuccessScreen                                                │
│ UBICACIÓN: src/transactions/TransactionSuccessScreen.tsx                          │
│                                                                                    │
│ USADA POR:                                                                         │
│   ✅ Swap (mismo chain y cross-chain)                                             │
│   ✅ Compra de Oro Digital (COPm/USDT → XAUt0)                                    │
│   ✅ Venta de Oro Digital (XAUt0 → COPm/USDT)                                     │
│                                                                                    │
│ COMPONENTES:                                                                       │
│   • Ícono: Celebration (64px) con fondo verde claro                               │
│   • Título: Personalizado según tipo (swap/goldBuy/goldSell)                      │
│   • Subtítulo: Mensaje de confirmación                                            │
│   • Detalles:                                                                      │
│     - De: TokenDisplay con monto y símbolo                                         │
│     - A: TokenDisplay con monto y símbolo                                          │
│   • Link: "Ver en el explorador" → WebViewScreen (Celoscan)                       │
│   • Botón: "Continuar" → navigateHome()                                            │
│                                                                                    │
│ PARÁMETROS:                                                                        │
│   - fromTokenId: string                                                            │
│   - toTokenId: string                                                              │
│   - fromAmount: string                                                             │
│   - toAmount: string                                                               │
│   - transactionHash: string                                                        │
│   - networkId: NetworkId                                                           │
│   - type: 'swap' | 'goldBuy' | 'goldSell'                                         │
│                                                                                    │
│ NAVEGACIÓN:                                                                        │
│   - noHeaderGestureDisabled (no se puede volver atrás con gesto)                  │
│   - navigateHome() al tocar "Continuar"                                            │
│                                                                                    │
│ EFECTOS:                                                                           │
│   - vibrateSuccess() al mostrar la pantalla                                        │
│                                                                                    │
│ TRADUCCIONES:                                                                      │
│   - transactionSuccess.swap.title                                                  │
│   - transactionSuccess.swap.subtitle                                               │
│   - transactionSuccess.goldBuy.title                                               │
│   - transactionSuccess.goldBuy.subtitle                                            │
│   - transactionSuccess.goldSell.title                                              │
│   - transactionSuccess.goldSell.subtitle                                           │
│   - transactionSuccess.from                                                        │
│   - transactionSuccess.to                                                          │
│   - transactionSuccess.viewOnExplorer                                              │
